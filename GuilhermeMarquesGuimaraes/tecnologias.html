Tecnologias utilizadas

Conheça as ferramentas utilizadas no desenvolvimento deste projeto.

HTML

HTML (HyperText Markup Language) é a linguagem utilizada para estruturar o conteúdo de páginas na web. Com ela é possível organizar títulos, parágrafos, imagens, links e diferentes seções de uma página, formando a base de qualquer site. O navegador interpreta essas estruturas para apresentar o conteúdo de forma organizada ao usuário.

📖 Aprender mais sobre HTML

CSS

CSS (Cascading Style Sheets) é a linguagem responsável pela aparência visual das páginas web. Ele permite definir cores, tamanhos de fonte, espaçamentos, bordas e layouts, transformando uma página simples em uma interface mais organizada e agradável para o usuário.

📖 Aprender mais sobre CSS

JavaScript

JavaScript é uma linguagem de programação utilizada para tornar páginas web interativas. Com ela é possível criar ações em botões, validar formulários, modificar conteúdos da página e responder às interações do usuário em tempo real.

📖 Aprender mais sobre JavaScript

VS Code

O Visual Studio Code é o editor de código utilizado para desenvolver este projeto. Ele oferece suporte para diversas linguagens, extensões e ferramentas que facilitam o desenvolvimento web.
